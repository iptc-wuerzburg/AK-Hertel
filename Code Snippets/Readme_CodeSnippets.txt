Code Snippetes sind kurze Teile von Code, welche für Wiederverwendung geeignet sind.
Generell sollten alle CodeSnippets sehr auführlich kommentiert sein, da sie möglichst leicht von anderen Nutzern verstanden und implementiert werden sollen.

AllWavesInFolder.ipf enthält ein Templat, welches erlaubt eine bestimmte Operation auf alle Waves im jetzigen data folder anwendet.
Background_Removal_Absorption.ipf enthält die Procedure, welche im Rahmen des Vortrags "Igor Progamming for Dummies" programmiert wurde.
